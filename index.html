Vulnerability Research

Real vulnerabilities found in open-source projects through code analysis

SSRF via User Webhook URL

High CVSS 7.5 | Open WebUI | CWE-918

Any authenticated user can configure a webhook URL in notification settings. The server makes unvalidated POST requests to that URL when calendar alerts fire, enabling access to cloud metadata endpoints and internal services.

DNS Rebinding in URL Validation

High CVSS 7.7 | Open WebUI | CWE-918

The SSRF protection resolves hostnames and checks for private IPs, but is vulnerable to DNS rebinding attacks. The developers acknowledge this in a code comment. Bypasses all URL validation protections.

XSS via Markdown-to-HTML Conversion

Medium CVSS 5.4 | Open WebUI | CWE-79

The /api/v1/utils/markdown endpoint converts user-supplied markdown to HTML using Python's markdown library without sanitization. Raw HTML passes through unchanged, enabling JavaScript injection.
