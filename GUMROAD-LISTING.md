## Description (for Gumroad product page)

**Stop guessing. Start auditing.**

WinSecAudit is a single PowerShell script that checks 50+ security configurations on any Windows Server — in under 60 seconds.

**No install. No dependencies. No license server. Just run it.**

### What it checks:
- Password & account policies (weak passwords, guest accounts, expired creds)
- Firewall configuration (disabled profiles, overly permissive rules)
- RDP exposure (NLA enforcement, default ports)
- SMB security (SMBv1/WannaCry vectors, signing, null sessions)
- Dangerous services (unquoted paths = privilege escalation)
- Patch status (days since last Windows Update)
- Open network ports (Risky listeners: Telnet, FTP, SNMP)
- Scheduled tasks (suspicious scripts, SYSTEM-level tasks)
- Audit logging (security log size and status)
- Windows Defender status

### What you get:
- **Dark-themed HTML report** — professional enough to hand to management
- **Security score (0-100)** with letter grade
- **Severity-ranked findings** — Critical down to Info
- **Remediation for every finding** — exact commands to fix it
- **Print-friendly** — export to PDF directly from browser

### Who it's for:
- IT admins who need a quick security baseline
- MSPs auditing client servers
- Consultants doing pre-engagement recon
- Anyone who needs to prove compliance

### Requirements:
- Windows Server 2016+ (or Windows 10/11)
- PowerShell 5.1+ (built-in, nothing to install)
- Run as Administrator

### What makes it different:
- **Zero dependencies** — pure PowerShell, no modules to install
- **Under 60 seconds** to complete
- **Actionable** — every finding has the exact fix command
- **Beautiful reports** — dark theme, responsive, print-ready

Built by a sysadmin, for sysadmins.
